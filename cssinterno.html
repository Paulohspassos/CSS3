Modo interno

Outra forma de incorporar o estilo CSS em nosso documento é o interno ou local.

Para se incorporar o CSS no modo interno inserimos o código de estilização dentro da tag <head> dessa forma podemos atribuir um estilo a toda a página.

Exemplo:

        
        <head>
            <meta charset="UTF-8">
            <style>
                body {
                    background-color: blanchedalmond;
                    font: normal 18px arial;
                }
            </style>
        </head>

Uma desvantagem é que o código fica muito extenso e a parte da css ocupará a maior parte do código, outra desvantagem é que o estilo será aplicado apenas ao documento em que foi escrito com isso, caso tenhamos a necessidade de aplicar a outras páginas teremos que copiar todo o código em cada documento.
