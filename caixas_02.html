Exemplo de caixa box-level

Parágrafos são exemplos de caixa box-level, mas os links são exemplo de caixas inline-level. Vamos ver como isso funciona.

A configuração CSS "display: block;", é a configuração do box-level

Para centralizar uma caixa a melhor maneira de centralizar uma caixa horizontalmente é a seguinte.

"margin: auto;"

Outline é uma linha que se encontra na area mais externa da nossa borda.

Para usar as medidas de forma resumida do padding devemos obedecer a seguinte ordem, 1-top, 2-right, 3-botton, 4-left. Caso queiramos repetir as medidas para todos os lados, basta coloca uma medida, quando colocamos dois valores o primeiro definirá top e botton e o segundo definirá right e left. Essa regra também vale para margin. Lembrando que usar o valor "auto", que define horizontamente os espaços, centralizará no eixo x nosso box.

As configurações de altura (height) e largura (width) só podem ser utilizados em caixa (box) que usem display block.
