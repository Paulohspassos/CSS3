Grouping Tags

As tags de agrupamento eram duas, a "div" e "span", hoje essa categoria de grouping tags estão associadas a semântica, ou seja, as tags "header", "main", "nav", "aside", "footer", "section" entre outros, com isso as tags se tornam mais descritivas facilitando assim o entedimento do que é esse elemento dentro do código.

Vamos montar um exemplo de organização sem as tags semânticas e com as tags semânticas.

Exemplo de código sem tag semântica

        
            <!DOCTYPE html>
            <html lang="pt-br">
            <head>
                <meta charset="UTF-8">
                <title>Sem tags semânticas</title>
            </head>
            <body>
                <div id="header">
                    <div id="nav">
                        <a href="#">Link</a>
                        <a href="#">Link</a>
                        <a href="#">Link</a>
                        <a href="#">Link</a>
                        <a href="#">Link</a>
                    </div>
                </div>
            <div id="main">
                <div class="section" id="assuntos">


                </div>
                <div class="section" id="noticias">
                    <div class="article">
                        <div class="aside">

                        </div>
                    </div>
                    <div class="article">
                        <div class="aside">

                        </div>
                    </div>
                </div>
            </div>
            <div class="footer">

            </div>
            </body>
            </html>

Sombras

Outro recurso muito interessante que podemos usar para dar mais beleza aos nossos projetos, as sombras dão um ar elegante e uma sensação 3d aos elementos da nossa página.

As sombras também utilizam das short-hands.

Box-shadow

Para configurar nossa box-shadow com short-hands os parâmetros são:

  1. Deslocamento Horizontal (valor positivo desloca para a direita e valor negativo desloca para a esquerda)
  2. Deslocamento Vertical (valor positivo desloca para baixo e valor negativo desloca para cima
  3. Blur
  4. Spread
  5. Cor da sombra

Existem as sombras externas "outset" e as sombras internas "inset".

É possivel insrir sombra em outras tags afinal todas as tags são caixas e podem ter sombras.

Bordas

As bordas também podem ser estilizadas e mudadas de acordo com o gosto do desenvolvedor e podemos fazer essa configuração pelo parâmento "border-radius:", a medida poderá ser em px para melhor controle, o controle short-hands também pode ser usado nesse parâmetro, com isso há a possibilidade de bordas diferentes para cada corner.

Bordas decoradas

Apesar de ser um recurso não muito apreciado pela maioria dos desenvolvedores e clientes, é sempre bom saber.

Através do parâmentro "border-image-source: url('pasta/nome_do_arquivo.png'), assim podemos utilizar qualquer imagem como borda de uma box.

Para ajustar a imagem podemos usar os parâmetros "border-image-slice:" e "border-image-repeat: repeat;".
