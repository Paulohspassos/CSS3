Seletores Personalizados

Para iniciarmos estudos de seletores personalizados vamos criar alguns títulos e parágrafos fictícios para dar volume ao nosso site.

Vamos usar uma folha externas de CSS.

Quando precisamos escolher algum elemento específico na nossa página para dar alguma carácteristica diferente das demais usamos seletores que as distingue das demais.

Nestes casos usamos o identificadore "id" na tag desejada dessa forma: <h1 id="principal"> assim quando formos estilizar apenas uma das tags h1, o estilo ou característica será aplicado só à aquela tag.

É aconselhável usar uma "id" à apenas uma tag específica evitando repetir a designação com o nome de uma id a outros elementos.

Na folha de CSS, para selecionarmos a tag com a id "principal" iniciaremos a seleção com o "#"

Para essa seleção na folha de CSS podemos usar as sintaxes: "h1#principal { }" ou " #principal { }"

Para se estilizar mais de um elemento utilizamos o "class", e para selecionarmos uma class usamos o ponto "." que funciona da mesma forma que a id com a sintaxe: "h2.corpo { }" ou ".corpo { }".

Podemos também aplicar mais de uma classe em um elemento para isso devemos usar a seguinte sintaxe: <p class="corpo basico"> assim ele será alterado por dois parâmetros diferentes.
