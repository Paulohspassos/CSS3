Caixas ou Box

Talves o elemento mais importante para se estudar, as caixas podem conter todo o conteúdo das nossas páginas, o estudo de "box model" é sugerido para uma maior evolução.

Vamos usar como exemplo o primeiro código que aprendemos o "Olá Mundo!", quando inserimos uma tag <h1> esse termo está dentro de uma caixa.

border é o limite da nossa caixa.

padding é o espaço entre o conteúdo e o limite.

margin é um espaço por fora do limite da nossa caixa, entre o conteúdo de uma caixa para outras caixas.

outline seria um limite do limite.

Tipos de Caixa

Box-level é um tipo de caixa que ocupará a largura total na janela do navegador.

Inline-level poderá ser inserido em linha com outros conteúdos.

Uma "div" é uma Box-level e uma "span" é uma Inline-level.
