Tipografia

Outro elemento muito importante em uma página bem desenvolvida são as fontes ou tipos, o tipo de fonte utilizado pode deixar a experiência do usuário bem mais confortável, por isso devemos ter bastante atenção a esse elemento.

Quando observamos bem a fonte vemos que algumas são mais agradáveis para se ler em determinados meios, como digital ou físico, do que outros, um tipo de letra é melhor para se ler em livros fisicos e outros tipos fica melhor para se ler em telas de tablets ou celulares, uns tipos é melhor para se ler artigo cientificos e outros tipos melhor para se ler uma charge, ou seja, devemos nos atentar ao tipo de fonte que fica melhor para que o usuario alvo tenha a melhor experiência possível.

Resumidamente, temos alguns elementos em cada tipo, fonte que favorencem cada finalidade, letras com serifa são projetadas para ser utilizadas impressas em papel, já as fontes san-serif são ideais para telas.

A fonte padrão dos navegadores é o Times New Roman.

Uma outra caracteristica importante é a fonte monoespaçada, com isso os espaços entre as letras não é influenciada pela letra em si, em fontes sem essa caracteristica algumas letras ocupam espaços diferentes o que pode gerar uma dificuldade à leitura.

Inserindo fontes

Quando queremos dar uma cara mais amigável ao nosso site, utilizamos o CSS para fazê-lo, com a propriedade "font", dessa forma há três forma de inserir esse estilo a nossa página, inline, quando o parâmetro "style" é inserido dentro da tag afetando assim apenas a tag a que está inserida, interno, quando criamos uma tag <style> na tag <head> do nosso documento e externo, por meio de outra folha de código com a extensão ".css" que será anexado por meio da tag <link>.

Para a insersão inline utilizamos a seguinte sintaxe:

        
            <p style="font-family: Impact, Haettenschweiler, 'Arial Narrow Bold', sans-serif;">Para a insersão inline utilizamos a seguinte sintaxe:</p>

Para se inserir através de configuração interna, a sintaxe ficará assim:

        
            <head>
                <meta charset="UTF-8">
                <title>Tipografia</title>
                <style>
                    body {
                            font-family: Arial, Helvetica, sans-serif;
                        }
                </style>
            </head>

E para inserir de forma externa utilizamos a tag link que ligará a nossa página a uma folha de estilo externa.

        
            <link rel="stylesheet" href="style.css">

            body {
                font-family: "san-serif";
            }

Tamanhos de fontes

Outra característica importanteda fonte é seu tamanho e suas representação.

As unidades de medida principais que podemos usar em nosso código são:

  • Medidas absolutas
    • cm
    • mm
    • in
    • px
    • pt
    • pc
  • Medidas relativas
    • em
    • ex
    • rem
    • vw
    • vh
    • %

Os mais recomendados principalmente em se tratando de adaptatividade é usar as medidas, relativa "em" e absoluta "px".

A fonte padrão nos navegadores é de 16px que é igual a 1em.
