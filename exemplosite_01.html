Site de Exemplo

Vamos criar um exemplo prático de estilização.

Configuração master

Para dar inicio as estilizações da nossa página primeiro devemos "resetar" alguns parâmetros que já vêm pré definidos nos navegadores.

Ao criarmos uma página, por padrão o navegador insere um padding(preenchimento) que cria um espaço entre o conteúdo seja texto ou outro elemento e os limites da janela do navegador, porém esse padding pode gerar algum problema na hora de dar um estilo personalizado a nossa página.

Para evitar esse problema, podemos iserir em nosso código CSS um seletor "* { padding: 0px; }, assim não haverá nenhum espaço indesejado em nossa página.

Outro parâmetro que podemos utilizar é o tipo de fonte (font-family), cor de fundo(background-color), entre outros parâmentros que serão mais gerais a nosso projeto.

O corpo da página

Outra area da nossa página muito importante é o corpo ou "body { background-image:linear-gradient(to right, #F1D8C9, #F2C3A7); }, que nesse caso criará um efeito degradé da esquerda para a direita

Esse código pode utilizar quantas cores quisermos para que o efeito seja de acordo com qualquer gosto.

Main ou principal

Neste exemplo que estamos vendo foi aplicado nas configurações de CSS os seguintes parâmetros:

            
                main {
                    background-color: white;
                    border-radius: 10px;
                    box-shadow: 5px 5px 15px rgba(136, 109, 93, 0.650);
                    width: 600px;
                    padding: 10px;
                    margin: auto;
                }
  • background-color: white; (com isso o fundo da caixa permanecerá branco.)
  • border-radius: 10px; (esse parâmetro deixará os cantos da caixa arredondados de acordo com a medida escolhida.)
  • box-shadow: 5px 5px 15px rgb(136, 109, 93, 0.650); (nesse parâmentro criamos um sobreamento em nossa caixa da seguindo a ordem da esquerda para a direita:
    1. medida da sombra das laterais em pixels, caso seja positivo a sombra estará a direita da caixa, caso seja negativo estará a esquerda;
    2. medida da sombra do topo e base da caixa, caso seja numero positivo estará embaixo, caso seja negativo estará em cima da caixa;
    3. medida de espalhamento da sombra, quanto maior o número mais "espalhada" será a sombra;
    4. cor da sombra, podemos usar todos os recursos de escolha de cores.
  • width: 600px; (essa medida representa a largura da caixa que estará fixada e limitada em 600 pixels.)
  • padding: 10px; (isso inserirá um espaço entre as palavras ou objetos como imagens e o limite da caixa.)
  • margin: auto; (que criará uma margem entre a caixa e o elemento pai, ou o corpo (body).)

Texto

Nas tags h1, h2, h3... estamos utilizando seletores personalizados para cada elemento, para isso basta criar o seletor referente ao elemento como:

            
                h1 {
                    color: #F39209;
                    text-align: center;
                    text-shadow: 1px 1px 2px rgba(129, 77, 5, 0.75);
                }

Parágrafos

Neste caso utilizamos o seletor p { text-align: justify; } para que o texto fique justificado na janela da página.

Além desse parâmetro, podemos utilizar qualquer outro para que nossa página fique de acordo com o gosto do contratante.

Para exemplificar o uso de gradiente (degradé) vamos inserir uma box com o background em greadiente, o código estará logo abaixo.

Foi criado uma div simples com o id="box" e as configuração css abaixo.

            
                #box {
                    width: 200px;
                    height: 20px;
                    border: solid 1px black;
                    border-radius: 5px;
                    background-image: linear-gradient(to right, blue, red);
                }

Esse foi um exemplo mais simples possível para se ter uma idéia de como inserir um efeito gradiente em nossas páginas.
