Pseudo Elementos

Os pseudo elementos são configurações que ajem nas classe, elementos, ids, etc, mexendo no coteúdo periférico do conteúdo.

Identificado por "::" e a ação desejada.

Exemplo:

  • Link 1
  • Link 2

A palavra [Link] não foi inserido no código HTML, foi inserido pelo pseudo elemento.

Quando formos selecionar algum elemento que está dentro de outro elemento usamos o simbolo ">" children (filho), assim a configuração estará imposta ao elemento que está dentro de outro.
