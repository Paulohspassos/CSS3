Variáveis em CSS

É possível utilizar variáveis em CSS facilitando assim o acesso a algumas configurações.

Para usar esse recurso devemos criar uma pseudo-classe chamada ":root" que é a raiz do nosso documento ou seja, tudo o que for inserido dentro dessa pseudo-classe servirá e poderá ser tuilizado por todo o documento que está anexado a folha/area de estilo.

Para isso devemos fazer o seguinte:

  1. Abrir a pseudo-classe através da declaração :root {....};
  2. Inserir "--nome_da_variável: valor_da_variável;
  3. Aplicar a variável através do código, "parâmetro: var(--nome_da_variável);".

Agora que as variáveis foram declaradas inseri-las ao nosso código.
