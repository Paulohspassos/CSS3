Site Responsivo

Responsividade nada mais é do que adaptar uma página para varios tamanhos de tela diferentes, para isso os ajustes são feitos no CSS.

Um dos recursos mais simples é limitando a largura mínima e máxima. Outra forma é utilizando porcentagem do tamanho da tela nas configurações do CSS, com isso a largura não estará bloqueada a uma medida mas se ajustará automaticamente conforme o tamanho da tela mudar.

Lembrando que a melhor maneira de se centralizar um elemento é delimitando uma dimensão de largura e adicionando uma margem com o valor "auto", assim o navegador ira manter o elemento no centro da página em qualquer exibição.

Para melhorar a responsividade da nossa página devemos usar os parâmetros "min-width" e "max-width", assim o navegador terá um limite minimo e máximo de largura mantendo a responsividade do site.

Para resolver de uma forma simples o width (largura) de fotos podemos usar o parâmetro "img { width:100%;}, assim a imagem será ajustada a janela do navegador.

Existem uma forma mais profissional de se tratar imagens em responsividade que é usando varios tamanhos de imagens, é necessário editar as imagens, porém fica mais profissional e elegante usar esse recurso.

Para isso devemos ter nossa imagem que estará em uma tag <img> dentro de outra tag, a tag <picture>, assim podemos usar o recurso <source media="(max-width: medida;)" srcset="endereço_da_imagem_com_outra_dimensão/imagem.ext">, assim a imagem será mudada automaticamente quando detectar uma tela com largura controlada, esse recurso, ao contrario do que foi dito anteriormente é tratado diretamente no documento HTML5 e não no CSS.

Um problema que encontramos foi centralizar uma imagem em nosso documento, pois o documento não é "display: block;", para sanar esse problema e centralizar nossas imgens devemos inserir esse parâmetro a uma img.
