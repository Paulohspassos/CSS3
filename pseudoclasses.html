Pseudo Classes

As divs são espaços reservados no código para que se insira outros elementos na nossa página, vem por padrão ocupando toda a largura (width)da página e ocupando a altura (height) necessária para o conteúdo da div e se ajusta almentando ou diminuíndo automáticamente.

As pseudo classes deverão estar relacionadas ao estado de um elemento específico ou classe e essa relação diz respeito ao estado desse elemento ou classe como se estão ativos "active", marcado "checked", vazio "empty", habilitado "enabled" entre outros, com isso podemos configurar esse estado.

Por exemplo o "hover" (tradução livre, flutuar, passar por cima) que pode ser configurado para mudar o estado ou fazer alguma ação determinada ao passar o mouse por cima de algum elemento, ao passar o mouse por cima do elemento ele mudará de cor nesse exemplo:

        
            div:hover {
                background-color: yellow;
            }

Outra pseudo classe que podemos usar como exemplo é o ":visited" que altera as propriedades de um link, para personalizar esses elementos por exemplo:

O código:

        
            a:visited {
                color: darkred;
            }

Outra pseudo classe é a ":active" que destaca a cor do link ativo em nossa página.
