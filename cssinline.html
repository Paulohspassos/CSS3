Básico CSS3

CSS são instruções de estilização, modelagem visual que o navegador utiliza para deixar nosso documento mais atraente ao usuário.

Podemos utilizar basicamente três formas de incorporar essa formatação ao documento inline, interno(local) ou externo.

inline

Essa forma de inserir o estilo visual inline deverá ser evitado sempre que possível pois deixa nosso código extenso e "sujo".

Essa atribuição, no entanto será sobreposto a todos os outros estilos.

Outro motivo é que ele se aplica somente a tag que estiver inserido, dessa forma caso tenhamos varias tags com o mesmo estilo, será necessário inserir manualmente o estilo em cada tag.

Exemplo:
<h1 style="color: rgb(14, 14, 56); background-color: rgb(174, 119, 226);/h1>
